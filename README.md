# Transit Tracker

Mobile-friendly astrology transit tracker — no API key needed, runs in the browser.

## Live App

[**Open on your phone →**](https://thomasingandello222-alt.github.io/transit-tracker/)

## Files

- `index.html` — Standalone phone-ready app using [Astronomy Engine](https://github.com/cosinekitty/astronomy). No API key, no backend, works offline after first load.
- `TransitTracker.tsx` — Original React component that uses the Anthropic Claude API (requires API key and server-side setup).

## Phone Use

1. Open the [live URL](https://thomasingandello222-alt.github.io/transit-tracker/) in Safari or Chrome on your phone.
2. Tap **Share → Add to Home Screen** for an app-like experience.

## Features

- Live planetary positions (tropical zodiac)
- Retrograde detection
- Aspects within 1.5° orb (conjunction, sextile, square, trine, opposition)
- Auto-generated TikTok-ready content script
- Dark/light mode, copy, and native share
